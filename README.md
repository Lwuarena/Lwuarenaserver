# BMC4 SERVER - Minecraft Java Required (Premium)
------------------------------
Before you can join the modded server, you have to install the mods on your client as well.
# Step 1
If you don't have curseforge downloaded on your computer yet, please do so using the following link https://www.curseforge.com/download/app
# Step 2
Create a minecraft profile and log in using your microsoft (Minecraft account) credentials
# Step 3 
Download the BMC4 modpack by going into the browser, and searching "BMC4"
